# Instructions

## Setup
cd into backend and frontend and run `npm install` in both directories.

## Start Backend
cd into backend and run `npm start`

## Start Frontend
cd into frontend and run `npm start`

## Run Tests
cd into backend and run `npm test`

## Notes
This example does not use any database, so the data is stored in memory. If you restart the server, the data will be lost.
The implementation is straightforward and does not use any advanced techniques. The focus is on the functionality and not on the architecture.
